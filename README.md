## Start whole project with database using docker

```bash
$ docker-compose up -d --build
```
## Also you can stop app container and use commands below to manage back separately

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Also you can find configurration files in src/config folder and .env file in root folder
